# Domain: Fullstack (全栈)

> 继承 `_base.md` + `frontend.md` + `backend.md` 全部内容。以下仅声明全栈领域增量。
> **合并协议**：当 frontend 与 backend 的规则冲突时，以主视角领域优先；无主视角时以更严格的规则为准。

## 领域增量编码纪律
- **前后端数据状态流转闭环**：前端提交 → 后端持久化 → 前端刷新，整条链路是否原子闭环？
- **接口契约一致性**：前端 TypeScript 类型定义与后端 API Response Schema 是否同步？

## 领域增量审查攻击向量
- **契约漂移**：前端调用的 API 字段与后端实际返回是否一致？是否存在前端依赖了后端未承诺的字段？
- **端到端安全**：前端输入校验是否与后端校验对齐？是否存在前端校验通过但后端绕过的路径？

## 纪律-攻击向量对偶映射 (领域增量)
| 编码纪律 (防守) | 审查攻击向量 (测试防守) |
|---|---|
| 前后端数据流转闭环 | 契约漂移：前端是否依赖后端未承诺的字段？ |
| 接口契约一致性 | 端到端安全：前端校验通过但后端绕过？ |
