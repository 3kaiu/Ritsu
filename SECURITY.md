# Security Policy

## Supported Versions

Currently, only the latest major version (v5.x) is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v5.x    | :white_check_mark: |
| < v5.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, send an email to security@3kaiu.com (placeholder).

We will acknowledge your report within 48 hours and provide a timeline for resolution.
