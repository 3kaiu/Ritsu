# Learning Engine Rules

- `scanHumanCorrections` is the shared entry point for scanning ctx diffs
- Heuristic rules are synthesized first, then LLM rules (if `RITSU_LLM_ENABLED=1`)
- LLM rules are generated by `llm-synthesizer.ts` via OpenAI-compatible API
- Vector search uses the Rust napi-rs engine with Jaccard fallback
